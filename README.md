Sobre o Módulo GIT do processo seletivo Hiring Coders 2021
por Bruno K J Momose

Inicialmente esse módulo foi um tanto confuso no qual tive que recorrer apoio à outras fontes de estudo.
Mas uma vez que obtive o conhecimento básico, seguir as aulas fluiu de maneira natural.
Uma recomendação para os próximos processos seletivos é que revise as aulas de maneira mais didatica explicando passo a passo cada comando, conceito...para que facilide o processo de aprendizagem do aluno.
